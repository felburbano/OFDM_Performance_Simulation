MATLAB implementation of an OFDM transceiver over a TDL multipath fading 
channel (3GPP TR 138.901). The system includes:

- QAM symbol generation (4 to 4096-QAM)
- Adaptive cyclic prefix length via energy accumulation criterion
- Pilot insertion with configurable periodicity 
- ML channel estimation from pilot sequences
- Zero-forcing frequency-domain equalization
- Symbol Error Rate evaluation
- Spectral analysis and constellation diagram visualization
